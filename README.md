# ConWaysteTheEnemy

Multiplayer Conway's Game of Life!


## Running the client

```
cargo run --bin client 127.0.0.1:9000
```

## Hacking

### Updating the life engine crate to the latest version in github

```
cargo update -p conway
```
